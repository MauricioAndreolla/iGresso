# 💻 Projeto 

> Aplicativo de compra de ingresso e review de filmes


# 🚀 Tecnologias

Esse projeto foi feito com as seguinte tecnologias:

- Javascript.
- NodeJS.

# 🏃 Como iniciar o projeto?

1. Abrir terminal do vscode

- npm install express   
- npm install ejs
- npm install bcryptjs 

2. Para executar a aplicação

-  node src/index.js
